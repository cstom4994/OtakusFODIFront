让 Visual Studio Code 用上毛玻璃效果

必要插件

由 EYHN 编写的 Vibrancy

食用方法

  1. 先切换为默认主题 Dark+
  2. 下载安装 Vibrancy
  3. 在 VSCode 中按下 F1 键输入 Reload Vibrancy
  4. 重启 VSCode 食用

设置

在 VSCode 设置中 找到 Extensions -> Vibrancy Effect

设置项 描述 设置
vscode_vibrancy.opacity 毛玻璃效果的透明度 范围: 0.0 ~ 1.0
vscode_vibrancy.theme 毛玻璃主题样式 Default Dark / Dark (Only Subbar)
vscode_vibrancy.imports 自定义 CSS 或 JS 文件的路径。
